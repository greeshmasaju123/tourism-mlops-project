# Tourism Wellness Package MLOps Project

This project predicts whether a customer is likely to purchase the Wellness Tourism Package.

## Required account setup

1. Replace \`YOUR_HF_USERNAME\` in the Python scripts and workflow file with your Hugging Face username.
2. Push the \`tourism_project\` folder to a GitHub repository.
3. Create a Hugging Face token with write access.
4. Add the token to GitHub repository secrets as \`HF_TOKEN\`.
5. Push to the \`main\` branch to run the GitHub Actions pipeline.
